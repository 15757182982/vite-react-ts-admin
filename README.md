# Vite-React-Ts-Amin

一个简易的vite-reat后台框架

## 项目简介

基于 vite-mobx-TypeScript-react 开发的后台管理系统

[示例链接](http://nestadmin_dt.jinxinapp.cn/login)

## 安装依赖

- node v12.14.0
- yarn 1.22.10

```bash
npm install
```

or

```bash
yarn
```

## 脚本描述

### 开发启动

```bash
npm run start
# mock模式启动
npm run start:mock
```

### 打包

```bash
npm run build
```

### 检查代码样式

```bash
npm run lint
```

### 测试代码

```bash
npm run test
```
